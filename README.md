# UAlbany Campus Portal (Static UI)

Static HTML/CSS UI for Albany Campus Portal platform. This project provides page layouts for authentication, profiles, posts, users and dashboard views, along with shared UI modules (navbar, chat). It is intended as a front‑end template and does not include a back‑end or API integration.

Features The project interactivity includes:

- Updating profile info and image
- Adding and Deleting user's own profile only
- Liking and unliking posts, commenting in posts
- Sign up and Sign in

Deployed via TBD [GitHub Pages]: (https://github.com/lindakovacs/UAlbany-Campus-Portal-Theme)

## Pages

- Home: index.html
- Auth: login.html, register.html
- Profile: profile.html, create-profile.html, add-education.html, add-experience.html
- Posts: posts.html, post.html
- Dashboard: dashboard.html

## Getting Started

Open any HTML file in your browser (for example, in VS Code, right-click on your HTML file (such as index.html) in the editor and select "Open with Live Server" from the context menu. Alternatively, you can use the keyboard shortcut Alt + L then Alt + O.). No build step or server is required for this Static UI website phase.
